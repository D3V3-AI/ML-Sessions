# ML Sessions

**ML Sessions** es un proyecto enfocado en realizar proyectos de Machine Learning e Inteligencia Artificial a través de problemas reales, resolviéndolos de principio a fin: desde el análisis exploratorio (EDA) hasta la implementación y uso del modelo.
La idea principal no es solo usar librerías, sino entender profundamente qué está ocurriendo detrás de cada modelo.

---

## 🎯 Objetivo

- Aprender Machine Learning de forma práctica
- Entender la matemática detrás de los modelos
- Implementar algoritmos desde cero cuando sea posible
- Construir una base sólida para proyectos reales de IA
